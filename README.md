## Quick start

You will need Rust and Cargo installed.

```
# Download the code
$ git clone https://github.com/m-peko/tetherion
$ cd tetherion

# Build the project in release mode
$ cargo build --release

# Run the Tetherion
$ ./target/release/tetherion
```
